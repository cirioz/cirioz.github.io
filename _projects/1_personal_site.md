A personal site / public notebook. Source on
[GitHub](https://github.com/cirioz/cirioz.github.io).

**Stack:** [Jekyll](https://jekyllrb.com/) + the
[al-folio](https://github.com/alshedivat/al-folio) theme, deployed via GitHub
Actions to GitHub Pages.

**Why:** I wanted one place for devlogs, tutorials, and weekly recaps across
programming, TouchDesigner, drawing, and music. Categories let me cross-cut
those topics by format (tutorial vs note vs weekly).
